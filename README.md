# Hemanth Bollam - Portfolio

Personal portfolio website for Bollam Hemanth Lakshmi Kumar, Machine Learning Engineer. 

## Features
- Modern, dark-themed UI with glassmorphism effects
- Fully responsive design
- Clean architecture with separate CSS, JS, and HTML files
- Sections for About, Resume, Education, Skills, Projects, and Achievements

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Setup
Simply open `index.html` in your browser to view the portfolio.
